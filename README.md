## Circuit wiring for code:
* **PS2 controller wireless reciever**
```
    - power (red):         3.3V (bus)
    - ground (black):      GND (bus)
    - data (brown):        P1.7 - 14
    - command (orange):    P1.6 - 15
    - Enable (yellow):     P2.3 - 34
    - clock (blue):        P6.7 - 35
```
* **IR Receiver**
```
    - G:                   GND (bus)
    - R:                   3.3V (bus)
    - Y:                   P5.1 - 33
```
